# nftables-firewall
Firewall IPv6 + IPv4

<pre>
- sudo nft -f /etc/nftables.conf       Charger la config<br/>
- sudo systemctl enable nftables       Activer au démarrage<br/>
- sudo systemctl start nftables        Démarrer le service<br/>
- sudo nft list ruleset                Vérifier les règles<br/>
</pre>
